# Form Inputs Demo

This example demonstrates all the new form input components in LuxeUI:

- **LuxeTextField** - Text inputs with floating labels
- **LuxeToggle** - Custom switches
- **LuxeCheckbox** - Checkboxes with animations
- **LuxeRadioButton** - Radio button groups
- **LuxeSlider** - Single-value sliders

## Running the Demo

### Option 1: Xcode
1. Open the `luxeUI` package in Xcode
2. Select the `FormInputsDemo` scheme
3. Run on your preferred simulator or device

### Option 2: Swift Package Manager
```bash
cd Examples/FormInputsDemo
swift run
```

## Features Showcased

- Floating label animations on text fields
- Smooth spring animations on toggles
- Haptic feedback on all interactions
- Gradient sliders with tap-to-seek
- Premium glassmorphism card containers
- Midnight theme preset

## Customization

All components support extensive customization through their Configuration structs. See the main API documentation for details.
