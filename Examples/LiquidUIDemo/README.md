# Liquid UI Demo

This app demonstrates the premium "Liquid UI" components of LuxeUI. 
These components focus on organic, fluid animations and "alive" interface elements.

## Showcase

### Liquid Loaders
- **Default**: Standard merging blob animation.
- **Gooey**: Enhanced blur threshold for stronger liquid effect.
- **Intense**: Fast-moving, multi-blob animation.

### Liquid Buttons
- **Interactive**: Backgrounds that morph and pulse.
- **Feedback**: Scale and haptic feedback on press.
- **Themes**: Neon, Lava, and Custom presets.

### Liquid Progress
- **Wave Animation**: Progress bars with fluid wave fill.
- **Smooth**: Animated transitions for progress updates.

### Decorator Blobs
- Background elements that morph and rotate to add life to the UI.

## Running the Demo

```bash
swift run LiquidUIDemo
```
