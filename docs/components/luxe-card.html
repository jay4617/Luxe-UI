LuxeCard

A premium floating glass card with hover effects, press animations, and haptic feedback. Perfect for dashboards, profiles, and content showcases.

Features

✓
Glass Effect

Frosted glass background with customizable blur and opacity

✓
Hover Animation

Scales up smoothly when hovered (macOS) or long-pressed

✓
Press Animation

Scales down when pressed for tactile feedback

✓
Haptic Feedback

Optional haptic response on interactions (iOS/macOS)

Basic Usage

Swift 
import LuxeUI

LuxeCard {
    VStack(spacing: 12) {
        Image(systemName: "star.fill")
            .font(.system(size: 40))
            .foregroundColor(.yellow)
        Text("Premium Feature")
            .font(.headline)
        Text("Hover to see the effect!")
            .font(.caption)
            .foregroundStyle(.secondary)
    }
    .padding()
}
.frame(width: 280, height: 180)

Configuration

Customize the card using LuxeCardConfiguration:

Swift 
let config = LuxeCardConfiguration(
    cornerRadius: 24,
    blur: 15,
    backgroundOpacity: 0.2,
    hoverScale: 1.05,
    pressScale: 0.95,
    enableHaptics: true
)

LuxeCard(configuration: config) {
    // Your content
}

Configuration Presets

.default

Balanced settings for most use cases

.compact

Smaller radius, subtle effects for dense layouts

.prominent

Large, bold appearance with stronger shadows

.subtle

Minimal, understated styling

.floating

Maximum elevation with deep shadows

Event Callbacks

Handle user interactions with modifier methods:

Swift 
LuxeCard {
    Text("Interactive Card")
}
.onTap {
    print("Card tapped!")
}
.onHoverStart {
    print("Hover began")
}
.onHoverEnd {
    print("Hover ended")
}

API Reference

LuxeCardConfiguration Properties

Property Type Default Description
cornerRadius CGFloat 20 Corner radius of the card
blur CGFloat 10 Blur radius for glass effect
backgroundOpacity Double 0.15 Opacity of background fill
borderWidth CGFloat 1 Border stroke width
shadowRadius CGFloat 20 Shadow blur radius
hoverScale CGFloat 1.02 Scale when hovered
pressScale CGFloat 0.98 Scale when pressed
enableHaptics Bool true Enable haptic feedback
