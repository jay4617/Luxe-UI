Refractive Glass

Featured

The 2026 signature effect—real lens distortion that physically warps the background like looking through water or curved glass.

✨
What makes this special?

Unlike standard glassmorphism (simple blur), Refractive Glass creates actual optical distortion with chromatic aberration, caustic light patterns, and multi-layer depth. This effect can't be easily replicated by hand—it's one of the main reasons developers use LuxeUI.

Features

✓
Real Lens Distortion

Warps the background at the edges like looking through curved glass

✓
Chromatic Aberration

RGB color separation at edges for realistic optical effects

✓
Animated Caustics

Light patterns that dance across the surface like sunlight through water

✓
Multi-Layer Depth

Stacked transparent layers create realistic glass depth

Basic Usage

Apply the refractive glass effect using the view modifier:

Swift 
import LuxeUI

VStack {
    Text("Premium Content")
        .font(.title)
}
.padding(32)
.refractiveGlass(intensity: 0.2)

RefractiveGlassCard Component

For a complete card experience with the effect pre-applied:

Swift 
RefractiveGlassCard(
    distortionIntensity: 0.25,
    chromaticAberration: true
) {
    VStack(spacing: 16) {
        Image(systemName: "wand.and.stars")
            .font(.system(size: 50))
        Text("Liquid Glass")
            .font(.title2.bold())
        Text("The 2026 signature effect")
            .foregroundStyle(.secondary)
    }
}
.frame(width: 300, height: 220)

Configuration

Use RefractiveGlassConfiguration for fine-grained control:

Swift 
let customConfig = RefractiveGlassConfiguration(
    distortionIntensity: 0.3,      // 0.0 - 1.0
    distortionRadius: 60,         // Points
    chromaticAberration: true,    // RGB split at edges
    aberrationStrength: 2.5,      // Intensity of color split
    causticAnimation: true,       // Animated light patterns
    causticSpeed: 0.4,            // Animation speed
    causticCount: 10,             // Number of light points
    layerCount: 4,                // Glass depth layers
    blurRadius: 25,               // Background blur
    cornerRadius: 28              // Corner rounding
)

MyContent()
    .refractiveGlass(configuration: customConfig)

Configuration Presets

Use built-in presets for common use cases:

.default

Balanced effect for most use cases

.subtle

Light effect for text-heavy content

.intense

Strong distortion for hero elements

.liquid

Water-like flowing distortion

.frosted

Heavy blur, minimal distortion

.minimal

Performance-optimized, fewer effects

Swift 
// Using presets
Content().refractiveGlass(configuration: .liquid)
Content().refractiveGlass(configuration: .subtle)
Content().refractiveGlass(configuration: .intense)

API Reference

RefractiveGlassConfiguration Properties

Property Type Default Description
distortionIntensity Double 0.2 Intensity of lens distortion (0.0 - 1.0)
distortionRadius CGFloat 50 Radius of the distortion effect in points
chromaticAberration Bool true Enable RGB color separation at edges
aberrationStrength CGFloat 2.0 Strength of chromatic aberration
causticAnimation Bool true Enable animated caustic light patterns
causticSpeed Double 0.5 Speed of caustic animation
causticCount Int 8 Number of caustic light points
layerCount Int 3 Number of glass layers for depth
blurRadius CGFloat 20 Background blur radius
cornerRadius CGFloat 24 Corner radius of the glass container
backgroundOpacity Double 0.15 Opacity of the glass background
enableHaptics Bool true Enable haptic feedback on interactions

Performance Tips

⚡
Performance Considerations

The refractive glass effect is GPU-intensive. For older devices or when using multiple glass elements:

  • Use .minimal or .frosted presets
  • Reduce layerCount to 1-2
  • Disable causticAnimation
  • Limit to 2-3 refractive elements per screen
