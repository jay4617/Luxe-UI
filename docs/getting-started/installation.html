Swift Package Manager (Recommended)

The easiest way to add LuxeUI to your project is through Xcode's built-in package manager.

Using Xcode

  1. Open your project in Xcode

    Navigate to File â†’ Add Packages...

  2. Enter the repository URL 
    https://github.com/Ronitsabhaya75/Luxe-UI
  3. Select version

    Choose "Up to Next Major Version" and enter 1.0.0

  4. Add to your target

    Select your app target and click "Add Package"

Using Package.swift

If you're developing a Swift package, add LuxeUI to your dependencies:

Package.swift 
import PackageDescription

let package = Package(
    name: "MyApp",
    platforms: [
        .iOS(.v15),
        .macOS(.v12)
    ],
    dependencies: [
        .package(
            url: "https://github.com/Ronitsabhaya75/Luxe-UI.git",
            from: "1.0.0"
        )
    ],
    targets: [
        .target(
            name: "MyApp",
            dependencies: ["LuxeUI"]
        )
    ]
)
